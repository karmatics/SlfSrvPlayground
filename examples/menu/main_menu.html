NOTE: This is an example use of the slfsrv-menu.js script, which builds on slfsrv to create menus like this one to launch both web applications and local applications from a single integrated menu. I use this menu on my own Macbook because I was tired of launching some tasks from the Macintosh Dock, others from my browser bookmarks, and yet others from Spotlight (in fact, this menu is now the ONLY item on my Dock; it is also associated with a hotkey for quick access at any time).

Because this is my personal menu, many of the links here likely won't work on your machine. Go ahead and copy this to your machine, select the "edit this" options, and make it your own.

My Favorite sections is "Don't look too often", where I keep a lists of distracting web sites that I don't want to be tempted to visit too often.

For more information about slfsrv-menu.js see slfsrv-menu Documentation.
